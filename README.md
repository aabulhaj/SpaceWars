# SpaceWars
A simple java implementation of a space war game.

```bash
Usage:
	java SpaceWars <spaceship types>

Available spaceship types:
	h - Human
	d - Drunkard
	r - Runner
	a - Aggressive
	b - Basher
	s - Special

You must supply at least two spaceship types, and the human type can only appear once.
```
